# 技术栈选择
| 层次 | 技术选型 | 说明 |
|------|----------|------|
| 框架 | Spring Boot 3.x, Spring Cloud | 微服务基础架构 |
| 语言 | Java 17+ | 核心开发语言 |
| 数据库 | MySQL 8.0 + Redis 7.0 | 关系型+缓存 |
| 向量DB | Milvus/Pinecone | 存储embeddings |
| 消息队列 | Apache Kafka/RabbitMQ | 异步通信 |
| 搜索引擎 | Elasticsearch | 全文检索 |
| 对象存储 | MinIO/S3 | 文件存储 |
| 监控 | Prometheus + Grafana | 系统监控 |
| 容器化 | Docker + Kubernetes | 部署运维 |
